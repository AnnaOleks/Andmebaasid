Andmetüüp:
1) Sümbol/Text (VARCHAR(25), CHAR(5), TEXT)
2) Arv (INT, SMALLINT, Decimal(5,2), kus 5 on - kõik arvud ja 2 - peale koma)
3) Loogika (bit, bool, boolean, true/falce)
4) Küüpäev (date/time, date, time)

-------------------------------------

Piirangud / ограничения
1) PRIMARY KEY - Это атрибут (или комбинация атрибутов), который уникально опраеделяет строку. Уникальность определения каждой строки в таблице
2) FOREIGN KEY - связь с первичным ключем другой таблицы
3) NOT NULL - не может быть пустым
4) UNIQUE - уникальность записи
5) CHECK - выбор

-------------------------------------

SQL - Structure Qery Language 
1) DDL - Data Definition Language - CREATE TABLE, ALTER TABLE, DROP TABLE
2) DML - Data Manipulation Language - INSERT INTO, UPDATE, DELETE
